# 🚍 BusGo – Modern Bus Ticket Booking UI Template

BusGo is a modern, responsive, and interactive bus ticket booking frontend template built using HTML, CSS, Bootstrap 5, and JavaScript.

Designed for frontend developers, beginners, and students who want a clean and professional travel booking interface for learning, portfolio projects, or customization.

---

## ✨ Features

* Fully Responsive Design
* Modern & Premium UI
* Smooth Scrolling Navigation
* Interactive Offers Carousel
* Popular Routes Section
* FAQ Accordion
* Contact Form UI
* Login & Signup Pages
* Hover Microinteractions
* Sticky Navbar
* Clean Folder Structure
* Easy To Customize

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
